JONATHAN HAMMOND

Product Designer based in Brighton England.

I design, prototype and build applications for desktop, mobile and film. Focusing on user Interaction design (IxD), User interfaces design (UI) and User Experience (UX).

ABOUT

I love using technology to create interactive performance tools. As an young seed I always wanted to perform with technology, not just operate. My goal is to design application that allow super human capibilties in the agumented real world.

When I speak about agumented human powers I’m talking about a digital layer of information, a operation overlay in the real world. Think back to when you first put headphones on… By listening to music in your own personal bubble, you’ve created your own augmented reality. My goal is to add control and informative data to this bubble using invisible technology. We live in a selfie world were user want to capture their adventures in the world, lets add a soundtrack to this.

What do I Create?

Designing user interactions via music control software is the best way to push hardware and user interfaces to the limits especially with new 3D control systems. Intuitive navigation and control is a must. Simple designs allowing informative interaction points with out the fustration. If I can’t control my own software to make reasonable sounds I’ve failed at user experience (UX)

What is User Experience (UX)

When I talk of UX I am refering to ‘user experience’. What we mean by this is to think metaganously about each interaction of a user and an interface.

What is Interaction Design (IxD)

When I talk of IxD im talking about behavior how the users actions reaction to the interface and how they respond to it.
